The enchantment can be obtained from the enchantment table, loot chests or trading with villagers.

## ⌨️ Commands

Give:

```mcfunction
/loot give @s loot timber_enchantment:timber
```

Enchant:

```mcfunction
/enchant @s timber_enchantment:timber
```
